# Images du site

Déposer ici les images optimisées du site.

- `oeuvres/` : photos des tableaux, idéalement en `.webp` ou `.avif`.
- `portrait-armelle.jpg` : portrait de l’artiste si disponible.
- `bleu-cendres-couverture.jpg` : couverture du recueil si disponible.
- `og-default.jpg` : image de partage 1200 × 630 px.

Tant que les chemins `image` sont vides dans `src/data/artworks.ts`, le site affiche un placeholder élégant.
